# Charts
Charts built on top of HTML5 canvas.

## Column chart
``` columnchart(ArrayX,ArrayY); ```

Eg: ``` columnchart(['Name','aa','bb','cc','dd','ee','ff'],['Marks',42,63,98,74,32]) ```

## Bar chart
``` barchart(ArrayX,ArrayY); ```

Eg: ``` barchart(['Name','aa','bb','cc','dd','ee','ff'],['Marks',42,63,98,74,32]) ```
